# HEDRA - Système de Génération Pédagogique

## Configuration de l'arrière-plan

Pour finaliser l'intégration de l'image d'arrière-plan Art Nouveau :

1. Sauvegardez l'image que vous souhaitez utiliser comme arrière-plan
2. Renommez-la en `background.jpg` (ou `background.png`)
3. Placez-la dans le dossier `images/` à la racine du projet

Le chemin final devrait être : `images/background.jpg`

L'image sera automatiquement appliquée en arrière-plan avec :
- Un effet de couverture complète (cover)
- Position centrée
- Overlay semi-transparent pour améliorer la lisibilité

## Structure du projet

```
Hedra/
├── images/
│   └── background.jpg  ← Placez votre image ici
├── index.html
└── README.md
```
